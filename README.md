# Astou Boutique — Maison, plage & élégance

Vitrine partenaire premium générée avec **DIGIY Projet Maison · BY DIGIYLYFE**.

Astou Boutique est une boutique située à Saly, Sénégal, spécialisée dans le linge de maison, les serviettes, les draps, les couvertures, les peignoirs, les serviettes pour hôtels, les foutas pour transats, les serviettes de plage, les robes, les tenues stylées, la confection et les crèmes de beauté.

Cette page sert à présenter la boutique, rassurer les clients, faciliter le contact WhatsApp direct, mettre en valeur les univers produits et connecter Astou Boutique à l’écosystème DIGIY MARKET.

---

## Objectif de la vitrine

La page permet aux visiteurs de :

- découvrir les produits et univers Astou Boutique ;
- contacter directement la boutique sur WhatsApp ;
- consulter la galerie visuelle ;
- comprendre rapidement l’offre pour la maison, les hôtels, les villas, les locations, la plage, la mode et la beauté ;
- accéder plus tard aux liens Google Maps, TikTok et DIGIY MARKET.
